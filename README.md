# Srandom

## Beschreibung
Die folgende Applikation "Srandom" erfüllt die Aufgabe, zufällige Lieder und Empfehlungen den Besucher vorzuschlagen und diese mit einem Klick für weiteres hören zur Verfügung zustellen.
